# ZOZ Skill: Building Modern Websites & Apps

This skill provides a comprehensive framework for creating modern, effective, and visually stunning websites and applications. It integrates the **WebForge Agent** methodology with the latest **2026 web design trends** to guide you through the entire creation process, from initial concept to final deployment.

## Core Principles

- **User-Centricity**: Every design and development decision should prioritize the user experience.
- **Performance by Default**: Build fast, efficient, and responsive websites that work seamlessly across all devices.
- **Accessibility First**: Ensure your creations are usable by everyone, regardless of their abilities.
- **Future-Proofing**: Embrace modern technologies and design trends to create websites that are built to last.

## The WebForge Agent Workflow

The WebForge Agent workflow is a 9-phase process that provides a structured approach to web development. This workflow ensures that all aspects of the project are covered, from initial research to post-launch maintenance.

| Phase | Title | Description |
|---|---|---|
| 1 | Research & Discovery | Understand the project goals, target audience, and competitive landscape. |
| 2 | Information Architecture | Plan the website's structure, navigation, and content hierarchy. |
| 3 | Wireframing | Create low-fidelity layouts to define the user interface and user flow. |
| 4 | Visual Design | Develop the website's visual identity, including color palettes, typography, and imagery. |
| 5 | Development | Build the website using modern web technologies. |
| 6 | Testing & Refinement | Test the website for functionality, performance, and usability. |
| 7 | Deployment | Launch the website to a live server. |
| 8 | Documentation & Handoff | Prepare documentation and hand off the project to the client or maintenance team. |
| 9 | Post-Launch Support & Maintenance | Monitor, maintain, and iterate on the website for the first 30 days after launch. |

For a more detailed breakdown of each phase, refer to `/home/ubuntu/skills/zoz/references/webforge_agent_phases.md`.
